# RUNTIME FC — Card Generator

런타임 FC 소셜 카드 생성 도구. 경기 예고 및 선발 명단 이미지를 1080×1080 PNG로 출력합니다.

## 기능

- **Match Announcement** — 경기 예고 카드 (상대팀 엠블럼 이미지 업로드 지원)
- **Starting XI A / B** — 선발 명단 카드 2종
- 선수 DB 기반 번호/이름 자동완성
- 1080×1080 PNG 다운로드 / 모바일 공유

## 기술 스택

- Next.js 15 (App Router)
- TypeScript
- html-to-image

## 로컬 실행

```bash
npm install
npm run dev
```
